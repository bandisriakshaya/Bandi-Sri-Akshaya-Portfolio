# Bandi Sri Akshaya - Portfolio

Simple HTML, CSS, and JavaScript portfolio website.

Live at: https://bandisriakshaya.github.io/Bandi-Sri-Akshaya-Portfolio/

## Files

- `index.html` - Main HTML file
- `styles.css` - All styles
- `script.js` - All functionality

## How to Use

Just open `index.html` in your browser. No build process needed!

Or use a simple server:
```bash
python -m http.server 8000
```

Then open http://localhost:8000

## Assets

Images and resume are in the `public/` folder.
